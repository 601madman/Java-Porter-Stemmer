Welcome to Java 7 Porter Stemmer

The Porter stemmer is widely known and used, but it crashes in Java 7. This project makes smaller modifications that allow it to work in Java 7. To easily download it, you can go to https://github.com/bean5/Java-Porter-Stemmer/zipball/master.
